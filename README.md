### Helpful git commands
Set your details
``` bash
git config --global user.name "John Doe"
git config --global user.email "john@example.com"
```

View remote urls

``` bash
git remote -v
```
Change origin url
``` bash
git remote set-url origin http//github.com/repo.git
```
